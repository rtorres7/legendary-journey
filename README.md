- Node + NPM (16.15.0 LTS - https://nodejs.org/en/)

## Project setup
```
npm run install-all-deps
```

### Running the application (server + clent)
```
npm run dev
```

### Running the frontend (client)
```
cd client
npm install
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
